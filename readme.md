# WinMux
Barebones terminal multiplexer for windows. Toy project.

### Usage

- Type `winmux` in a terminal to start multiplexing (automatically creates a session).
- Use `Ctrl` `Alt` `Ins` to create a new session.
- Use `Ctrl` `Alt` `BackSpace` or close the session window to destroy the active session.
- Use `Ctrl` `Alt` `PgUp`/`PgDown` to navigate back and forth between sessions.
- Use `Ctrl` `Alt` `Home` to detach.
- Use `Ctrl` `Alt` `End` to attach.
